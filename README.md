# Задания 2 и 3

Полный комплект файлов по теме **Электронного дневника медицинского самоконтроля (ЭДМС)**.

## Задание 2

Выполнено проектирование базы данных для системы ЭДМС.

### Что входит

- проектирование реляционной БД;
- ER-модель;
- описание таблиц и атрибутов;
- SQL-схема;
- наполнение тестовыми данными;
- полный SQL-дамп;
- SQL-запросы для отчетов;
- результаты отчетов в `csv` и `txt`;
- готовая база SQLite.

### Файлы задания 2

- `docs/schema_description.md` - описание таблиц;
- `docs/er_model.md` - ER-модель в Mermaid;
- `docs/er_diagram.png` - ER-диаграмма в виде изображения;
- `sql/01_schema.sql` - создание таблиц;
- `sql/02_data.sql` - заполнение тестовыми данными;
- `sql/03_queries.sql` - все отчеты одним файлом;
- `sql/04_dump.sql` - полный дамп БД;
- `sql/01_indicator_dynamics.sql` ... `sql/03_top_actions.sql` - отдельные запросы;
- `reports/output/*.csv` и `reports/output/*.txt` - результаты запросов;
- `db/edms.db` - готовая база SQLite.

## Задание 3

Выполнены макеты пользовательских интерфейсов для проектируемой системы ЭДМС.

### Что входит

- макет ведения справочника действий самоконтроля;
- макет ввода новой записи самоконтроля;
- макет формирования отчетов пользователя;
- пояснения, какие объекты БД используются на каждом экране;
- HTML-прототип экранных форм для просмотра в браузере.

### Файлы задания 3

- `docs/task3_ui_mockups.md` - описание экранов и связь элементов интерфейса с таблицами БД;
- `docs/task3_ui_mockups.html` - статический HTML-прототип макетов интерфейса.

## Предметная область

База данных отражает работу системы ЭДМС: хранение пациентов и врачей, фиксацию действий и симптомов, запись объективных показателей самоконтроля, а также историю формирования отчетов.

## Отчеты

1. Динамика объективного показателя за выбранный период.
2. Три даты наиболее плохого самочувствия по показателю «Самочувствие».
3. TOP-3 действий по средним значениям пульса и давления.

## Структура репозитория

- `docs/` - документация, ER-модель и макеты интерфейса;
- `sql/` - схема, данные, запросы и дамп;
- `db/` - файл базы данных SQLite;
- `reports/output/` - результаты выполнения запросов.
